---
title: Check Monitor Status
api: GET /monitor/{reportId}
description: Check if a specific repo is being monitored
---

## Path Parameters

<ParamField path="reportId" type="string" required>
  Report ID to check.
</ParamField>

<RequestExample>
```bash cURL
curl http://localhost:3001/monitor/a1b2c3d4-e5f6-7890-abcd-ef1234567890
```
</RequestExample>

<ResponseExample>
```json 200
{
  "success": true,
  "data": {
    "monitoring": true
  }
}
```
</ResponseExample>

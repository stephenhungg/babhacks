---
title: Get Market
api: GET /market/{marketId}
description: Retrieve market data including bets and consensus
---

## Path Parameters

<ParamField path="marketId" type="string" required>
  Market ID.
</ParamField>

## Response

Returns the full market state including all bets and consensus valuation.

<ResponseField name="data" type="ValuationMarket">
  See [Create Market](/api-reference/market-create) for the full `ValuationMarket` schema.
</ResponseField>

<RequestExample>
```bash cURL
curl http://localhost:3001/market/m1a2b3c4-d5e6-7890-abcd-ef1234567890
```
</RequestExample>
